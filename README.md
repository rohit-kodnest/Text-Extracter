# Image Text Extractor

This project allows you to extract text from images using Optical Character Recognition (OCR) capabilities provided by Tesseract.js.

## How to Run

1. Open `index.html` in a web browser.
2. Select an image file and click on "Extract Text".
3. See the text extracted from the image displayed on the page.

For development, make sure `tesseract.js` is installed via npm:

npm install
And then serve or open the `index.html` in your browser.
